# oracao-automacao-worker
Worker gratuito para geração diária de vídeos 16:9 (8 min) dos canais de oração (PT/EN/ES/PL). Lê scripts e ordens de serviço no Google Drive, renderiza com FFmpeg/gTTS, salva MP4 e thumbnails no Drive e registra logs. Orquestrado por planilha (Apps Script) + GitHub Actions.
